読了時間: 1分

正規表現テスター

/ /

使い方

  1. 正規表現パターンを入力
  2. フラグを設定（g=全体, i=大小文字無視, m=複数行）
  3. テスト文字列を入力するとリアルタイムでマッチ箇所がハイライト

用途例

  • メールアドレス・電話番号の検証パターン確認
  • ログからの情報抽出
  • テキスト置換の事前確認
